# time-double-ledger-SIM-10-zones
Time Ledger Sim MVP (Go+Rust). Production-flavored backend for a "time-currency" ledger with:  Double-entry ledger, 10 worldwide zones with operator-controlled status, Fraud/ops incidents per zone, messaging with NATS JetStream + Transactional Outbox + Inbox dedup Observability: structured logs, Prometheus metrics, OpenTelemetry traces (Jaeger)
